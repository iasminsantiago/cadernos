# COMPUTAÇAO


SISTEMA OPERACIONAL = software // software que é intermediario entre usuário e hardware do pc
  - traduz a entrada do usuário (linguagem pelo teclado, clique no mouse) para a linguagem de máquina e a daída/resposta do hardware para o usuário. É UMA INTERFACE.
  - todo aplicativo instalado no meu celular ou pc ... é instalado SOBRE o SO
  - Gerencia: todos os processos do pc pra executar tarefas, organizar as demandas, gerencia e aloca memória, gerencia dispositivos conectados (mouse, teclado etc) e arquivos
  - Funções: segurança -- impedir tarefas etc, detectar/monitorar erros -- gera log, agendar tarefas
  - Recursos: de segurançam interface do usuário, de utilitários e serviços de sistema, suporte a desenvolvimento de aplicativos (eu codando)
  - Componentes: kernel e shell, são o que executa o sistema operacional
    -   Kernel: Componente central do SO. Trata das interaçoes entre hardware e SO. Todos os outros componentes do SO dependem dele pra rodar. Funções: gerencia entrada-saida, memória, execução dos apps, dispositivos, chamdas do sistema
    -   Shell: trata das interações do usuário, camada mais externa, gerencia A interação entre usuário e SO. FunçÕes: solicita entradas dos usuários, interpreta entrada do usuario pro SO, manipula saida do SO

WINDOWS - SO padrão da microsoft, usa linguagens C, C++ e assembly.
LINUX - seu Kernel foi escrito por Linus Tovald. Open-source = pode ser editado e distribuido, SEM pagamento de royalties
VirtualBox - software de virtualização. 
  Virtualizaçao = executar SO sobre outro SO, consumindo recursos deste de forma isolada. É uma aplicaçào que faz FUNÇAO de virtualização
