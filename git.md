## GIT E REPOSITORIES

- git status - mostra o que nao ta staged, o que ain da nao foi colcoado num pacote / commit
- lista colorida - ainda não estão no commit, git status mostrará eles
- se ja rodei git commit, os arquivos coloridos ja sairam da lista colorida e do git status: foram jogados num PACOTE com numero id na branch
- git show - mostr ao que tem no commit
